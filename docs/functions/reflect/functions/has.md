# Function: has()

> **has**(`target`, `propertyKey`): `boolean`

Equivalent to `propertyKey in target`.

## Parameters

• **target**: `object`

Object that contains the property on itself or in its prototype chain.

• **propertyKey**: `PropertyKey`

Name of the property.

## Returns

`boolean`
